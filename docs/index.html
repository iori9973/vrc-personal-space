Personal Space

他プレーヤーが近づくと自分だけが離れてパーソナルスペースを保つ VRChat 用ツールの VPM リスティングです。

VCC に追加

ボタンが動かない場合は、VCC の Settings → Packages → Add Repository に次の URL を貼り付けてください：
https://iori9973.github.io/vrc-personal-space/index.json

追加後、アバタープロジェクトの Manage Packages から「Personal Space」を Install できます。 別途 osc/ の Python アプリの起動が必要です（README 参照）。
