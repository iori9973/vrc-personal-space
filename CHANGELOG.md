# Changelog

## [0.7.0]

- **機能を単体で選べるように**。押し出し機能と透明化機能をそれぞれ独立したチェックボックスにし、**透明化だけを ON**（OSC アプリ不要のアバター単体運用）や**押し出しだけ**のセットアップが可能になった。
  - Setup Window の「機能」欄で `押し出し機能（要OSCアプリ）` / `透明化機能（近づかれたら消える）` を個別に切り替え。オフにした機能のセンサー・パラメータ・メニュー項目は生成されない。
  - セットアップは毎回いったん全生成物を撤去してから、有効な機能だけを作り直す方式に統一（設定変更が確実に反映される）。
  - メニューはオンにした機能の項目だけを追加（透明化のみなら「透明化」トグルだけ）。
- **範囲表示を塗りつぶし円 → リング（輪）に変更**。足元の反応範囲を、中が抜けた輪郭のリングで表示するようにした（自分視点の視界を塞ぎにくい）。

## [0.6.1]

- 範囲表示/透明化のセットアップ時に `MissingComponentException`（PS_RangeViz の MeshFilter 等）で失敗するバグを修正。`GetComponent() ?? AddComponent()` が Unity の疑似 null を無視する問題を `if (== null)` に置換。

## [0.6.0]

- **透明化モード**を追加。近づかれると**他人視点で自分のメッシュが消える**（自分視点では見えたまま）。メニュー「透明化」ON で有効。押し出しの代替として使える。
  - 仕組み: 近接検知の Contact Receiver → 同期パラメータ `PS_Near` → リモート(IsLocal:false)で全 Renderer を非表示。OSC アプリ不要のアバター単体ギミック。
  - 制約: VRChat の仕様上、特定の人だけに消えることはできず**全員から消える**。

## [0.5.0]

- **センサーを縦カプセル化**。高さ方向を広く覆うため、身長差のあるアバターでもすり抜けにくくなった（水平距離ベースで検知）。
- **反応範囲の視覚表示**を追加。足元に半透明ディスクを表示し、メニュー「範囲表示」ON で `PS_Range` に連動して大きさが変わる（自分のみに表示・ローカル）。

## [0.4.1]

- 「削除」ボタンで `PS_ModularAvatar`(MA コンポーネント一式)と生成メニュー資産(PS_Menu/PS_MenuRoot)が消えなかったバグを修正。空の MA Parameters が残って統合オブジェクトが削除されない問題も解消し、完全にクリーンアップするように。

## [0.4.0]

- **1ウィンドウ・1ボタンに統合**。Setup Window の「セットアップ / 更新」だけで、センサー＋遅延補償＋メニューを一括生成するように変更（旧「Remote Offset」ウィンドウは廃止）。
- 上級者向けに「遅延補償を含める」「メニューを追加する」のチェックボックスで個別オフも可能。

## [0.3.1]

- 旧バージョン(〜0.2.0)がアバターの Expression Menu に直接差した「Personal Space」submenu を、セットアップ時に自動で掃除するように修正（`CleanupLegacyMenu`）。これがないと未定義パラメータ警告やメニュー重複が残っていた。

## [0.3.0]

- **Modular Avatar 対応（非破壊注入）**。パラメータ・メニュー・遅延補償アニメを、Expression 資産を直接編集する代わりに MA コンポーネント（MA Parameters / MA Menu Installer / MA Merge Animator）でアバター直下の `PS_ModularAvatar` に注入するように変更。
  - **FaceEmo など NDMF 系ツールと共存**でき、ビルド時に項目が消える問題を解消。
  - リモート位置ズラしの Controller が**自動で FX にマージ**される（手動 Merge Animator 設定が不要に）。
  - メニューは「Personal Space」サブメニューとして追加。
  - Modular Avatar (>=1.10.0) が必須依存に。

## [0.2.0]

- 方向センサーの既定を 16 に変更（最大反応半径の既定も 1.5m に）。
- Expression Menu に実行時調整の Radial を追加：反応範囲 `PS_Range` / 押し出しの強さ `PS_Gain` / 遅延補償の量 `PS_Lead`（いずれも非同期、アプリが購読して反映）。
- 反応範囲は物理半径を最大にしたまま近接度しきい値で縮める方式（アニメ生成不要）。

## [0.1.0]

- 初回リリース。
- N 方向 Contact Receiver による他プレーヤーの接近検知（Local Only, ランク非計上）。
- OSC アプリによる自動回避（逃走ベクトル合成 → `/input` 移動）。
- OSCQuery による自動接続（固定ポートへフォールバック可）。
- リモート位置ズラし（遅延補償）。
- Expression Menu による ON/OFF。
